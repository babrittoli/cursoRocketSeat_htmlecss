# Funções

Emprogramação, funções são reconhecidas por causa um reaproveitamento de código
* Nome seguido de abre e fecha parentesis
* Recebe argumentos

## Exemplos 
- rgb()
- hsl()
- url()
- calc()

``` Css
@import url("http://urlaqui.com/style.css");

{
    color: rgb(255,-0,100);
    width: calc(100% - 10px);
}

```
